# bookmarklet

## このページについて
自作のブックマークレットをおいとくリポジトリです

## ブックマークレットについて
### table-row-to-clipbord.txt
クリックしたテーブルの列をクリップボードにコピーします。  

### table-row-to-clipbord-no-alert.txt
table-row-to-clipbord.txtとできることは一緒です。  
アラートがいちいち表示されるのがめんどくさい人用。
